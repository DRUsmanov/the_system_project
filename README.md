# Проект находится в разработке!

# Описание
Проект по разработке системы управления производственными цехами для Газпром нефтехим Салават.

# Модули
+ __Табель учета рабочего времени__
  
  Основной модуль необходимый для работы остальных, фиксирует фактическое пристуствие работников на рабочем месте. Предусморено версионирование табеля между несколькими работниками - намеренное создание конфликтов для поддержание истинности данных.
+ __Модуль контроля обучения__
  
  Отслеживает сроки обучения работников, сопостовляет их с табелем учета рабочего времени, составляет график обучения на год.
+ __Модуль контроля инструктажей__
  
  Отслеживает сроки проведения инструктажей работников.

+ __Модуль оборудования__

  Структурирует оборудование производственного цеха.

+ __Модуль графика плановых производственных работ__

  Формирует графики плановых производственных работ. Ведет учет выполненных работ.
